# Ground Claude Plugin Marketplace

AI-powered marketing workflows for DTC and ecommerce teams.

## Available Plugins

### Ecommerce Marketing Skills
13 AI workflow systems covering UGC campaigns, SEO content, and affiliate program management.

## Install

**Step 1: Add this marketplace**
```
claude plugin marketplace add github:Sheidmirj/claude-marketplace
```

**Step 2: Install the plugin**
```
claude plugin install ecommerce-marketing-skills
```
